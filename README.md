# Stopwatch

# Description
- This is a simple stopwatch that allows users to start, stop, and reset a timer.

# Features
- Star stopwatch button
- Stop/pause stopwatch button
- Displays minutes and seconds

# Tools used
- HTML
- CSS
- JavaScript

# Links used
- https://pixabay.com/

# Challenge/Learned
- A challenge for me was figuring out my syntax errors that was preventing the stopwatch to start.
- I learned how to create a functioning stopwatch and learned from my errors.
